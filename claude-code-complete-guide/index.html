🤖 Claude Code 是什麼？

🚀 簡單說

Claude Code 是 Anthropic 出的 AI 程式碼助手，你可以用對話方式叫它幫你寫程式、查資料、自動化工作。

📋 基本操作

💻

終端機開啟

在終端機輸入 claude 就能開始對話

📝

下指令

用自然語言描述你要什麼，它就會幫你做

📁

檔案操作

可以讀寫檔案、執行命令、搜尋程式碼

⚡

自動化

設定 hooks 和 skills 讓它自動完成重複工作

💡 為什麼要學 Claude Code？

  • ✅ 免費（用 API Key 或 Pro 訂閱）
  • ✅ 本機運行，資料不外洩
  • ✅ 可外接各種工具（MCP）
  • ✅ 可自動化任何重複工作

🔌 MCP 是什麼？

🌉 MCP = Model Context Protocol

簡單說就是讓 AI 可以接外部工具的橋樑。

Claude Code 本身只能讀寫檔案、跑終端機，裝了 MCP server 之後，它就能做更多事：

  • 📂 查文件
  • 📓 讀 Notion
  • 🌐 操作瀏覽器
  • 🎨 生成圖片

🛠️ 安裝 MCP 的方式

# 方式一：直接用 claude mcp add claude mcp add 伺服器名稱 -- 指令 # 方式二：透過 npx 安裝 claude mcp add nanobanana-mcp -- npx -y @ycse/nanobanana-mcp -e GOOGLE_AI_API_KEY=你的key

⚠️ 注意

每次新增 MCP 後需要重開 Claude Code才能使用新工具。

🎨 用 AI 生成 IG 圖文

🍌 NanoBanana MCP

把 Google Gemini 的圖片生成模型接進 Claude Code，等於讓你的 AI 助手多了一隻會畫圖的手。

📥 安裝步驟

1️⃣

申請 API Key

去 aistudio.google.com 申請 Gemini API Key

2️⃣

安裝 MCP

在 Claude Code 輸入安裝指令（見下）

3️⃣

重開軟體

關掉重開，讓新工具生效

4️⃣

開始生圖

用指令生圖、修圖、對話微調

⌨️ 安裝指令

claude mcp add nanobanana-mcp -- npx -y @ycse/nanobanana-mcp -e GOOGLE_AI_API_KEY=你的key

🎯 可用工具

  • 📐 set_aspect_ratio — 設定圖片比例
  • 🔄 set_model — 切 flash（快）或 pro（精細）
  • 🎨 gemini_generate_image — 生圖
  • ✏️ gemini_edit_image — 改圖
  • 💬 gemini_chat — 多輪對話微調

✨ 進階：IG Carousel Skill

作者還寫了一個 /ig-carousel Skill，只要打一句話：

/ig-carousel Claude Code 教學 — 科技綠暗底

它就會自動：

  • ✅ 設定 1:1 比例
  • ✅ 中文翻英文 prompt
  • ✅ 套上品牌排版
  • ✅ 清除 C2PA metadata（IG 不會顯示「Made with AI」）

📝 適用場景

適合： 快速出圖、風格一致的系列圖、每天要發文但沒時間開設計軟體

不太適合： 像素級精準排版、品牌設計稿、複雜文字佈局（AI 文字偶爾會有錯字）

📦 Everything Claude Code (ECC)

🎯 什麼是 ECC？

就是前人幫你踩完雷的攻略說明書！

這是一個 50K+ stars ⭐、Anthropic Hackathon 冠軍的開源套件，把各種好用技能、規則、自動化脚本整理好，讓你直接套用。

📊 ECC 內建什麼？

🤖

38 Agents

專業子代理（程式碼審查、架構設計、TDD 等）

⚡

156 Skills

工作流定義（Python、Go、Django、Laravel 等）

⌨️

72 Commands

指令（/plan、/tdd、/security-scan 等）

📐

34 Rules

語言規範（TypeScript、Python、Go、Swift、PHP）

🔑 核心功能

  • 🧠 連續學習 — 自動從對話中提取模式變成技能
  • 💾 記憶持久化 — 跨 session 自動保存/載入上下文
  • 🎯 Token 優化 — 模型選擇、Prompt 精簡
  • 🛡️ 安全掃描 — AgentShield 漏洞檢測
  • 🔄 Continuous Learning — 自動從錯誤中學習

📥 安裝方式

# 方式一：Plugin 方式（推薦） /plugin marketplace add affaan-m/everything-claude-code /plugin install everything-claude-code@everything-claude-code # 方式二：手動安裝 git clone https://github.com/affaan-m/everything-claude-code.git cd everything-claude-code ./install.sh --profile full

🔗 官方連結

ECC 官網：ecc.tools

GitHub：github.com/affaan-m/everything-claude-code

💡 為什麼要用 ECC？

就像玩遊戲有人幫你寫好攻略，ECC 把：

  • ✅ 別人踩過的坑 → 變成 Rules
  • ✅ 常用工作流 → 變成 Skills
  • ✅ 最佳實踐 → 變成 Agents
  • ✅ 自動化脚本 → 變成 Hooks

你不用從頭摸索，直接站在巨人的肩膀上！

💵

/context

/compact

🧹

/clear

💬

/btw

💎
