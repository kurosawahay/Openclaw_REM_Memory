DeepSeek 技能優化配置

aliases: DeepSeek, 技能優化, 模型配置, 效能提升

📋 優化背景

根據 Gemini 的建議，為提升 DeepSeek 在 OpenClaw 的工作效能，特別配置以下核心技能。

🎯 五大核心技能配置

1. 📖 jina-reader (網頁轉 Markdown)

  • 核心目的：解決 DeepSeek 對複雜 HTML 原始碼處理效率低的問題
  • 落地應用：配合「蒼影」Agent，將新聞網頁直接轉為結構化 Markdown
  • 預期效果：減少 30% 以上的摘要幻覺
  • API 金鑰：[JINA_KEY]
  • 免費版限制：每月 10M tokens (約 10,000 頁)

2. 🔍 tavily-search (實時聯網搜尋)

  • 核心目的：補足 DeepSeek 訓練資料的時間差（2024 中葉後的資訊）
  • 落地應用：針對「影音市場趨勢」與「AI 工具更新」，獲取 2026 年當下的實時情報
  • 預期效果：避免資訊過時，確保決策基於最新情報
  • API 金鑰：[TAVILY_KEY]
  • 免費版限制：每月 1,000 次搜尋，每次最多 5 個結果

3. 📁 filesystem (進階檔案管理)

  • 核心目的：支撐「記憶揉捏」架構的物理穩定性
  • 落地應用：負責在 /memory/ 目錄下精準執行子專案檔案（sub_*.md）的創建、檢索與 Append 動作
  • 技術基礎：clawdbot-filesystem v1.0.2
  • 功能特色：多層級目錄管理、智能過濾、批次處理

4. ✂️ context-optimizer (上下文剪枝)

  • 核心目的：優化 128K 窗口的讀取品質並節省 API 成本
  • 落地應用：自動識別並刪除對話中重複的指令、無效的過場廢話
  • 預期效果：確保 DeepSeek 每一點算力都花在「精華訊息」上
  • 技術基礎：npm 依賴 (tiktoken, @xenova/transformers) 已安裝

5. 🐍 python-executor (已移除)

  • 原始目的：賦予 Reasoner (R1) 模型實體驗證能力
  • 移除原因：inference.sh 需要付費，訪客帳號餘額不足
  • 替代方案：使用本地 Python 環境或尋找免費替代方案
